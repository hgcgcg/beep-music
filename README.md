# BEEP-MUSIC
这是hgcgcg的一些beep音乐，你可以在你的个人电脑上运行它。  
你需要安装软件包beep <br>
  
Debian/Deepin/Ubuntu/Linux Mint

~~~
sudo apt install beep
~~~

RHEL/CentOS/Fedora

~~~
sudo yum install beep
~~~

Archlinux/Manjaro

~~~
pacman -S beep
~~~
